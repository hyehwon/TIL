### 오늘 한 일
- 데잇걸즈 Git/GitHub 기초 강의를 들었다.
    - 리눅스 기본 커맨드라인을 사용해봤다.
        - pwd(print working directory)
        - cd(change directory)
        - ls(list) : 옵션 -a, -al (ls -a, ls -al)
        - clear : 화면 깨끗하게
        - history : 작성한 명령어 다시 보기
            - !number : history에 있는 명령어 행번호로 이전에 작성한 명령어를 다시 실행할 수 있다
    - vim으로 파일을 수정하고 저장했다.
        - 입력 모드 전환 : 키보드 i 
        - 명령 모드 전환 : 키보드 esc
        - 저장하고 vim 빠져 나오기 : :wp
    - Git Bash에서 디렉토리를 생성하고 저장소와 연결하는 것을 해봤다.
